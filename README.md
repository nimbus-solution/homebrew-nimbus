# homebrew-nimbus

Official Homebrew tap for **Nimbus** — run Salesforce Apex tests locally, without an org.

No Docker. No sandbox. Results in seconds.

---

## Install

```sh
brew tap nimbus-solution/nimbus
brew install nimbus
```

## Upgrade

```sh
brew upgrade nimbus
```

## Verify

```sh
nimbus --version
```
